HTML

  • HTML stands for Hyper Text Markup Language
  • The head element contains info about the webpage
  • The body element represents the visible content shown to the user
  • Attributes define additional information about an element
  • Metadata is data about our webpage that the browser uses to effectively serve our webpage to users

CSS

  • CSS stands for Cascading Style Sheets
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content outside an element
  • Seelectors define the element or attributes to which the rules will apply to
  • CSS sheets need to be linked to HTML sheets in order to work correctly
  • CSS box model

Git

  • Github lets us share our code with others, making collaboration with other developers easier
  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
